# Introduction

Mannaiah is a NestJS-based backend application designed to manage contacts and synchronize orders from WooCommerce. It leverages modern technologies to provide a robust and scalable solution for e-commerce management.

## Core Features

- **MongoDB Storage**: Utilizes MongoDB for flexible and scalable data storage.
- **Automated Tasks**: Uses cron jobs to perform periodic synchronization and maintenance tasks.
- **Secure Authentication**: Integrated with Logto for secure, JWT-based authentication and role-based access control.
- **External Sync**: Seamlessly syncs orders from external tools like WooCommerce, converting them into contacts and maintaining data consistency.
